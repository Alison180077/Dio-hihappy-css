Propriedadestext-overflow

A propriedade text-overflow no CSS é usada para indicar como o conteúdo de texto deve ser exibido quando ele ultrapassa o tamanho do contêiner. Ela é normalmente usada em conjunto com overflow e white-space para criar efeitos como corte de texto com reticências.
